
# Marco Teórico

El dataset con el que se quiere trabajar consiste de 4 series de tiempo, cada una consiste de 15 puntos equispacidos registrados. Cada serie representa un tratamiento aplicado a una muestra inicial de granos de kéfir.

Lo que buscamos nosotros es encontrar parametros $\theta^*$ de uno de los modelos que minimice el error cuadrado promedio de los datos (MSE).
